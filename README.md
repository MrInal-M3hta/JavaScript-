🚀 JavaScript Project

A simple JavaScript project demonstrating core concepts of JavaScript, DOM manipulation, and modern ES6 features.

This project is created to practice and understand JavaScript fundamentals and how JavaScript interacts with HTML and CSS.

---

📌 Features

- Beginner friendly code
- Clean and simple UI
- DOM Manipulation
- Event Handling
- ES6 JavaScript
- Responsive design

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
